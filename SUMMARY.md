## 🚀 사용 방법

### 1. 설치 및 실행
```bash
npm install
npm run dev
```

### 2. API 키 설정
`.env.local` 파일에 다음을 추가합니다:
```env
OPENAI_API_KEY=your_key_here
```

### 3. 개발 시작
- `INTEGRATION_PLAN.md`를 확인하여 통합 계획을 검토하세요
- `FEATURE_COMPARISON.md`를 확인하여 기능 비교를 검토하세요
- `QUICK_START.md`에서 빠른 시작 방법을 확인하세요
