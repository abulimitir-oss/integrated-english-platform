# 🌟 통합 영어 학습 플랫폼

7가지 최고의 영어 학습 플랫폼 기능을 통합한 AI 기반 영어 학습 애플리케이션입니다.

## 🎯 프로젝트 개요

이 플랫폼은 다음의 주요 기능을 통합합니다:

1. **WriteFlow** - AI 글쓰기 보조 (작문 보조, 채점, 교정)
2. **CoTalk** - 상황별 대화 연습
3. 음성 입력 및 발음 교정 기능
4. 커뮤니티 기능 및 게임화 요소
5. CEFR 레벨 테스트 및 상황별 학습 콘텐츠
6. 말하기·작문·읽기 통합 훈련
7. 메타버스 기반 3D 게임화 학습 경험

## ✨ 핵심 기능 모듈

### 📝 1. 글쓰기 모듈
- ✍️ 실시간 문법 교정: AI가 문법과 어휘 오류를 감지
- 📚 작문 채점: 에세이 피드백 제공
- 🎨 문장 다듬기: 가독성과 표현력 개선
- 💡 일일 문장 수집: 학습 소재 제공

### 💬 2. 대화 모듈
- 🌍 다수의 상황별 대화 시나리오
- 🎤 음성 입력: 실시간 음성 인식
- 🔊 발음 교정: 발음 분석 및 피드백
- 🎮 메타버스 환경(선택): 몰입형 대화 경험

### 📖 3. 읽기 모듈
- 시험형 독해 연습 (TOEIC/TOEFL/SAT 등)
- 시간 제한 및 즉시 채점
- 문제 풀이 전략 제공

### 🎤 4. 말하기 모듈
- 녹음 연습 및 발음 분석
- AI와의 실시간 대화 연습
- 유창성 및 발음 피드백

### 📚 5. 어휘 모듈
- 간격 반복 학습(Spaced Repetition)
- CEFR 레벨 기반 어휘
- 어휘 테스트 및 단어장 기능

### 📊 6. 평가 모듈
- CEFR 레벨 테스트
- 학습 분석(회화 능력, 어휘 다양성 등)
- 강점/약점 분석 및 맞춤형 추천

### 🎮 7. 게임화 모듈
- 레벨/XP 시스템
- 과제 및 퀘스트
- 버디 시스템 및 리더보드

### 👥 8. 커뮤니티 모듈
- 학습 후기 공유
- Q&A 및 토론 포럼
- 챌린지 및 활동

### 🌍 9. 다국어 지원
- 한국어, 영어, 일본어, 중국어 등

## 🛠️ 기술 스택

## 🚀 빠른 시작

1. 의존성 설치
```bash
npm install
```

2. `.env.local` 생성 (필요 시)
```env
OPENAI_API_KEY=your_openai_api_key_here
GROQ_API_KEY=your_groq_api_key_here
```

3. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 http://localhost:3000 접속
